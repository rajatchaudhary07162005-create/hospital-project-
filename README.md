# hospital-project-data-analysis 
I built a hospital management database from scratch and wrote 10 SQL queries to extract real business insights from the data.

📊 Here's what I analyzed:
✅ Total number of patients across all hospitals
✅ Average doctors available per hospital
✅ Top 3 departments with the highest patient count
✅ Hospital with the maximum medical expenses
✅ Daily average medical expenses per hospital
✅ Longest patient hospital stay
✅ Total patients treated per city
✅ Average length of stay per department
✅ Department with the lowest patient count
✅ Monthly medical expenses report

🔧 Key SQL concepts used:
→ Aggregate functions (SUM, AVG, ROUND)
→ GROUP BY, ORDER BY, LIMIT
→ Date arithmetic (Discharge Date – Admission Date)
→ TO_CHAR & EXTRACT for date formatting
→ Subqueries & casting with ::NUMERIC

This project helped me understand how SQL can turn raw hospital data into actionable insights for healthcare management.
